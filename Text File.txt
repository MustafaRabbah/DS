Q12 Zero Trust

B. Never trust, always verify

Q15 Zero Trust
Which Zero Trust principle ensures users receive only the minimum permissions
needed?
A. Micro-Segmentation
B. Phishing-Resistant MFA
C. Continuous Authentication
D. Least Privilege Access
ANSWER D. Least Privilege Access

Q16 Zero Trust
What does Micro-Segmentation achieve in a Zero Trust model?
A. Encrypts all traffic between servers
B. Monitors user behaviour in real time
C. Isolates network segments to contain breaches
D. Applies biometric authentication to all users
ANSWER C. Isolates network segments to contain breaches
Micro-Segmentation isolates network segments so that if one is compromised, the breach cannot easily
spread to other parts of the network.
